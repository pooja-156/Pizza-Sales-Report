# Pizza-Sales-Report
# 🍕 Pizza Sales Dashboard

## 📌 Project Title
**Pizza Sales Analysis Dashboard**

## 📖 Short Description / Purpose
This interactive Power BI dashboard provides a comprehensive overview of pizza sales data to help business stakeholders analyze performance trends, customer preferences, and operational efficiency. The goal is to derive actionable insights that support data-driven decision-making in a retail food environment.

## 🛠️ Tech Stack - List of Technologies Used
- **Power BI**: For building interactive visual dashboards
- **DAX (Data Analysis Expressions)**: For data modeling and calculations
- **Power Query (M Language)**: For data transformation and preprocessing
- **Microsoft Excel / CSV** (assumed): For storing raw data
- **Date Table (Calendar Table)**: For enabling time intelligence functions

## 🗃️ Data Sources
- Internal POS system export (e.g., Excel or CSV files containing:
  - Pizza Orders
  - Order Details
  - Pizza Types and Sizes
  - Order Timestamps
)
- Manually created or imported **Date/Calendar table** for time-based analysis

## ✨ Features / Highlights
- 📈 **Sales Overview**: Total revenue, average order value, and total quantity sold
- 🕒 **Time-Based Trends**: Monthly and daily sales performance visualization
- 🍕 **Top Selling Pizzas**: Ranking of pizzas by quantity sold or revenue
- 📍 **Category-wise Breakdown**: Sales by pizza type, size, and category
- 🧠 **Interactive Filters**: Slicers for pizza type, date range, and size
- 🔍 **KPI Cards**: Highlight key metrics at a glance
- 📊 **Visuals Used**: Line charts, bar charts, pie charts, and KPI indicators
